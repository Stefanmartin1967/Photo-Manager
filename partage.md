# Réponses concernant le partage et le renommage

## 1. Renommage des photos
C'est fait ! Désormais, lorsque vous cliquez sur le nom d'une photo ou d'un groupe, **tout le texte est automatiquement sélectionné**. Vous n'avez plus besoin d'effacer manuellement le texte, vous pouvez taper directement le nouveau nom.

## 2. Partage sur PC (Facebook / Messenger)
Comme vous l'avez supposé, le comportement dépend de l'appareil :

*   **Sur Mobile (Android / iOS) :** Le bouton "Partager" ouvre le menu natif de votre téléphone. Si vous avez les applications Facebook ou Messenger installées, elles apparaîtront dans la liste et vous pourrez partager les photos directement.
*   **Sur PC :** Les navigateurs (Chrome, Edge, Firefox) ne permettent généralement pas de partager des fichiers directement vers des sites web comme Facebook ou Messenger via le menu "Partager" du système. C'est une limitation technique des navigateurs sur ordinateur.
    *   *Solution sur PC :* Il faut télécharger les photos (Bouton "Sauvegarder") puis les glisser-déposer manuellement dans votre conversation Messenger ou sur votre mur Facebook.

Conformément à votre demande ("Si oui [ça marche sur mobile], ne change rien"), **aucune modification n'a été apportée au code de partage**. La fonctionnalité reste standard et compatible avec les mobiles.
